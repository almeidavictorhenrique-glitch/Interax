# 🧠 Totem InteracX / FlexMedia

Projeto de um totem interativo inteligente com uso de Inteligência Artificial, desenvolvido para melhorar a experiência de visitantes em ambientes como zoológicos.

---

## 🚀 Funcionalidades

* 🎤 Interação por voz e texto
* 🌍 Suporte a múltiplos idiomas
* 🔊 Respostas em áudio (acessibilidade)
* 📊 Coleta de métricas de uso
* 🤖 Integração com APIs (IA, tradução, etc.)
* 🖥️ Interface interativa e intuitiva

---

## 🧩 Tecnologias utilizadas

* Python
* Flask
* HTML, CSS e JavaScript
* APIs de IA
* Raspberry Pi (integração futura com hardware)

## ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/almeidavictorhenrique-glitch/Interax.git
```

2. Acesse a pasta:

```
cd totem_front_end
```

3. Instale as dependências:

```
pip install flask
```

4. Execute o projeto:

```
python app.py
```

5. Acesse no navegador:

```
http://localhost:5000
```

## 🎯 Objetivo do projeto

O objetivo é criar uma solução autônoma que:

* Melhore a experiência do visitante
* Reduza a necessidade de suporte humano
* Gere dados estratégicos para gestão
* Seja escalável para diferentes ambientes
